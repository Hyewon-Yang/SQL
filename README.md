# SQL

SQL이란, 관계형 데이터베이스 시스템(RDBMS)에서 자료를 관리 및 처리하기 위해 설계된 언어다.  
목표로 하는 문제에 대한 인사이트를 데이터로부터 얻기 위해서는, 우선 적절한 데이터 수집 및 전처리가 필요하다.  
SQL은 이 단계에서 데이터베이스에 저장되어 있는 데이터에 접근하고 추출하는데 사용된다.
